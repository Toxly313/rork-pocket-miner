# Dorffenster auf Vollbild bringen und Ruckler beim Bewegen beheben

**Features**
- [x] Das Dorffenster füllt den gesamten verfügbaren Bildschirm aus.
- [x] Wischen und Zoomen im Dorf werden deutlich flüssiger und reagieren direkter.
- [x] Die Darstellung wird so entschlackt, dass die starken Hänger beim Starten und Beenden von Bewegungen verschwinden.
- [x] Die aktuelle Projektbasis wird auf die gewünschte Version vereinheitlicht.

**Design**
- [x] Das Dorf wirkt größer und immersiver, weil keine unnötigen Ränder mehr Platz wegnehmen.
- [x] Die Oberfläche bleibt klar und mobilfreundlich, damit Ressourcenleiste und Navigation trotzdem gut erreichbar sind.
- [x] Bewegungen sollen sich stabil, leicht und ohne sichtbare Sprünge anfühlen.

**Pages / Screens**
- [x] **Dorf**: Vollbild-Ansicht für die Basis mit optimierter Bewegung und Zoom.
- [x] **Geöffnetes Gebäude**: Bleibt weiterhin als eigener Inhalt sichtbar, nutzt aber ebenfalls die neue große Fläche harmonisch aus.
